ID e CLASS

ID

o ID funciona como um CPF assim como posso só ter um numero de CPF, cada elemento só pode ter um valor de ID. O CPF é unico, ou seja, ninguem pode ter um CPF igual ao meu. isso signifa qque o ID é unico e não pode repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. diferente do ID em que você sò pode ter um valor, no CLASS você pode ter varios valores. tambem diferente do ID, em que não se pode se repetir um valor, no CLASS você pode utilizar o mesmo valor para varios elementos.
